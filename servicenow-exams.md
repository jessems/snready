# ServiceNow Certification Exams

A comprehensive list of all ServiceNow certification exams.

---

## Mainline Certifications

### Foundation Level

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CSA** | Certified System Administrator | Foundation certification validating ability to configure, implement, and maintain the ServiceNow platform |

### Developer Level

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CAD** | Certified Application Developer | Validates skills in developing applications on the ServiceNow platform, including scripting, workflows, and custom application development |

### Architect/Expert Level

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CTA** | Certified Technical Architect | Advanced certification for technical architects designing enterprise ServiceNow solutions |

### Platform Owner

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CPOA** | Certified Platform Owner Associate | Validates understanding of platform governance, strategy, and management |

## Certified Implementation Specialist (CIS) Certifications

### IT Service Management

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CIS-ITSM** | CIS - IT Service Management | Implementing ITSM applications (Incident, Problem, Change, Request Management) |

### Customer Service

| Exam Code | Certification Name | Description |
|-----------|-------------------|-------------|
| **CIS-CSM** | CIS - Customer Service Management | Implementing customer service solutions and case management |

### Human Resources

## Micro-Certifications

ServiceNow also offers micro-certifications for specific skills and features. These are smaller, focused certifications that validate expertise in particular areas:

- Flow Designer Micro-Certification
- Virtual Agent Micro-Certification
- Predictive Intelligence Micro-Certification
- IntegrationHub Micro-Certification
- Mobile App Builder Micro-Certification
- Service Portal Micro-Certification
- And many more specialized topics

*Note: Micro-certifications are frequently updated. Check [ServiceNow Learning](https://learning.servicenow.com) for the current list.*

## Important Notes

1. **Prerequisites**: Most CIS certifications require passing the CSA exam first
2. **Version Updates**: Exams are updated with each major ServiceNow release (Vancouver, Washington DC, Xanadu, etc.)
3. **Recertification**: Certifications require maintenance/recertification to stay current
4. **Delta Exams**: When new releases come out, delta exams allow certified professionals to update their credentials
